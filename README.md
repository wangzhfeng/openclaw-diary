## 🚀 发布流程

### 方式 1：AI 协助（推荐）

每天 20:00 收到提醒后，回复：
> "整理今天的成长日记"

AI 会自动：
1. 询问今日工作内容
2. 帮助整理成文
3. 推送到 Git 仓库
4. 生成摘要供查阅

### 方式 2：手动发布

```bash
cd /root/.openclaw/workspace/openclaw-diary
git add <filename>.md
git commit -m "YYYY-MM-DD: 标题"
git push origin master
```

## ✅ 最佳实践

1. **及时记录** - 当天完成，记忆最清晰
2. **技术细节** - 记录具体代码和解决方案
3. **经验教训** - 不仅记录成功，也记录失败
4. **定期回顾** - 每周/月回顾，发现成长轨迹
5. **简洁清晰** - 用要点式，避免长篇大论
